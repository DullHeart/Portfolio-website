Introduction

Python is a high-level, interpreted, and general-purpose programming language known for its readability and simplicity. It is cross‑platform and supports multiple programming paradigms—including procedural, object‑oriented, and functional programming. Python’s design emphasizes code clarity and brevity, making it a popular choice for beginners as well as professionals working in web development, data science, automation, and beyond. Inside its runtime environment (for example, a terminal, an IDE, or even a web-based interpreter), Python connects with an extensive standard library that supplies modules and tools for tasks such as file I/O, regular expressions, networking, and more.

What you should already know

This guide assumes you have the following background:

  • A general understanding of computer programming concepts.
  • Familiarity with concepts like variables, control flow (loops and conditionals), and functions.
  • Basic experience with using a command-line interface.

If you’re completely new to programming, you might want to check out introductory tutorials on Python before diving into more detailed topics.

Python and other languages

Python differs from many other languages in several key areas:

  • Dynamic and Strong Typing:

    Unlike statically typed languages (e.g., Java or C++), Python requires no explicit type declarations. However, Python is strongly typed—operations that mix incompatible types will raise errors.

  • Whitespace Significance:

    Instead of using curly brackets to define code blocks, Python uses indentation to determine block structure. This design choice leads to code that is both visually uncluttered and semantically clear.

  • Interpreted Nature:

    Python code is executed line-by-line by an interpreter. This allows for rapid development and testing.

  • Extensive Standard Library:

    Python includes modules for many tasks—from web services to data manipulation—which means that many common tasks require minimal external code.
Hello world

To get started with Python, open your favorite editor or the interactive Python shell (REPL) and type the following code:

print("Hello, World!")

This simple program outputs the text “Hello, World!” to the console.

Variables and data types

Python uses dynamic typing; you create a variable simply by assigning a value to a name. Variable names must begin with a letter (or underscore) and can include numbers thereafter. Python is case‑sensitive.

# Declaration and assignment x = 42 name = "Alice" pi = 3.14159 # You can change the type of a variable at any time: x = "Now I'm a string"

Python supports several built-in data types:

  • Numbers:

    Integers (int), floating‑point numbers (float), and complex numbers (complex).

  • Booleans:

    True and False.

  • Strings:

    Text enclosed in single, double, or triple quotes.

  • Sequences:

    Lists (mutable), tuples (immutable), and ranges.

  • Mappings:

    Dictionaries for key/value pairs.

  • Sets:

    Unordered collection without duplicate elements.

For more details on Python’s built‑in types and their behaviors, see the Python Glossary on MDN.

Comments
  • Single-line comments in Python start with the # symbol: # This is a single-line comment. print("Comments help explain code!")
  • Multi-line comments aren’t supported in the same way as in some other languages; however, you can use consecutive # lines or use triple‑quoted strings (which are actually string literals that can serve as docstrings): """ This is a multi-line comment that can span several lines. """
Control flow statements

If...Else Statements

Use an if statement to execute code only when a condition is true, and use elif and else for additional conditions:

temperature = 75 if temperature > 80: print("It's hot outside!") elif temperature < 60: print("It's cool outside!") else: print("It's just right!")

Loops

For Loop

Python’s for loop iterates over any iterable (such as a list, tuple, or string):

# Iterating over a list fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(fruit) # Using range() to iterate over numbers for i in range(5): print(i)

While Loop

The while loop repeatedly executes a block as long as a condition is true:

n = 0 while n < 3: print("n is", n) n += 1 # Increment n by 1
Functions

Functions

Functions are defined using the def keyword. They encapsulate code for reuse and can accept parameters and return values.

def square(number): """Return the square of a number.""" return number * number print(square(5)) # Output: 25

Python also supports anonymous functions via the lambda keyword for simple expressions:

double = lambda x: x * 2 print(double(10)) # Output: 20

For more on functions and best practices (including docstrings), see the official Python Tutorial

Variable scope and constants

  • Global vs. Local Variables:

    Variables defined outside of functions are global. Variables defined inside a function (or block) are local to that function.

    x = "global" def func(): x = "local" print("Inside function:", x) func() # Prints: local print("Outside:", x) # Prints: global

  • Constants:

    Python does not enforce constants. By convention, names written in all capital letters indicate that they should not be changed.

    PI = 3.14159
Data structures

Lists and Tuples

  • Lists are mutable sequences:

    numbers = [1, 2, 3] numbers.append(4) print(numbers) # Output: [1, 2, 3, 4]
  • Tuples are immutable sequences: coordinates = (10, 20) # coordinates[0] = 15 # This would raise an error

Dictionaries

Mappings: Dictionaries for key/value pairs

person = {"name": "Alice", "age": 30} print(person["name"]) # Output: Alice # Adding a new key/value pair: person["city"] = "New York"

Sets

Sets are unordered collections without duplicates:

unique_numbers = {1, 2, 2, 3} print(unique_numbers) # Output: {1, 2, 3}
Classes and object-oriented programming

Python supports object‑oriented programming with classes. Define a class using the class keyword and use __init__ to initialize objects

class Dog: def __init__(self, name, breed): self.name = name # Instance variable self.breed = breed def bark(self): print(f"{self.name} says: Woof!") # Create an instance of Dog my_dog = Dog("Buddy", "Golden Retriever") my_dog.bark() # Output: Buddy says: Woof!

Every method in a class receives self as its first parameter to refer to the current instance.

Modules and packages

Python code is organized into modules (files with a .py extension) and packages (directories containing an __init__.py file). Import modules using the import keyword:

import math print(math.sqrt(16)) # Output: 4.0

For more details on modules, see the official Python documentation

Exception handling

Handle errors using try , except, else, and finally blocks:

try: result = 10 / 0 except ZeroDivisionError: print("Cannot divide by zero!") else: print("Division succeeded!") finally: print("This is executed no matter what.")
